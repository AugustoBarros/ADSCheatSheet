    Propriedades e atalhos

    O que a propriedade display define?
    Define de qual maneira o elemento será disposto na tela.

    Qual a diferença entre Float e Clear
  • Float define onde o elemento HTML será posicionado, movendo-o para um ponto específico.
  • Clear se aplica para "limpar" os elementos abaixo dos elementos com Float

    • O que define a propriedade float?
    O posicionamento do elemento para o ponto mais próximo.

    XHTML

    É a combinação de XML com HTML.

    Qual é a sua principal finalidade do XHTML?

    Uma vantagem do XHTML?
    XHTML possui compatibilidade do seu código com futuras aplicações de usuários.

    XHTML pode ser definido como:
    Um processo de padronização de páginas HTML
